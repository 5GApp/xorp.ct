XORP Source Code Access

XORP is managed in a git repository hosted by github.com. For more information on git, see the git home page.

Bug reports, code contributions, and comments for applications in the XORP git repository should be sent to the appropriate mailing list.

Anonymous (Read-Only) Access to the XORP git Repository

Visit the XORP github project page and click on the 'Git Read-Only' link. Then, clone the specified URL:
git clone git://github.com/greearb/xorp.ct.git

Troubleshooting

Any comments or problems with access to XORP should be sent to greearb@candelatech.com and/or the mailing list.
