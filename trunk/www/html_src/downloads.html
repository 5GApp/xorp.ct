XORP Source Downloads

As of March 2006, the architecture is basically complete and it might be used as the basis for research. The software is reasonably stable and can be used for routing, but there are known issues. See the current status of the different parts of XORP, or the release notes for information about changes between releases.

Please note that to build all the XORP modules and test programs with debugging enabled, you will need nearly 1.4GB of free disk space. XORP is written in C++, and has been mainly developed on FreeBSD 4.x. The current release 1.2 is known to compile with gcc 2.95.x, 2.96, 3.3.4, and 3.4.2 20040827 (prerelease). Building XORP for Windows requires the use of the Minimalist GNU for Windows (MinGW and MSYS) toolchain.

The code should build and run on Linux 2.4.x, Linux 2.6.x, FreeBSD, NetBSD, OpenBSD, MacOS X, and Windows Server 2003.
