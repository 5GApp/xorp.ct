XORP Source Downloads

As of July 2008, the architecture is basically complete. The software is stable and can be used for routing. See the current status of the different parts of XORP, or the release notes for information about changes between releases.

Please note that to build all the XORP modules and test programs with debugging enabled, you will need nearly 1.4GB of free disk space. XORP is written in C++, and has been mainly developed on FreeBSD. The current release 1.4 is known to compile with gcc. Building XORP for Windows requires the use of the Minimalist GNU for Windows (MinGW and MSYS) toolchain.

The code should build and run on Linux, DragonFlyBSD, FreeBSD, NetBSD, OpenBSD, MacOS X, and Windows Server 2003.

If you have problems downloading, try a mirror site.
